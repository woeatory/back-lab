## Installation

```bash
git clone https://github.com/woeatory/back-lab.git
```

```bash
# installing dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

Application will be accesed on <http://localhost:3000/>

To see swagger documentation go to <http://localhost:3000/api#>

Manage database

```bash
npx prisma studio
```

## Docker

```bash
docker-compose build
```

```bash
docker-compose up
```
