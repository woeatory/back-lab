## Installation

```bash
git clone https://github.com/woeatory/back-lab1.git
```

```bash
# installing dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

Application will be accesed on <http://localhost:3000/>

To see swagger documentation go to <http://localhost:3000/api#>
